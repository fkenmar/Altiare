# Altiare

Stardew Valley inspired isekai RPG game that's work in-progress

Built in **Godot 4.6** (GDScript)

Every character or monster is procedurally generated but follow the same structure

## Layout

The Godot project lives in [`new-game-project/`](new-game-project/).

```bash
godot --path new-game-project       # play (opens on the intro cutscene)
godot -e --path new-game-project    # open in the Godot editor
```

See [`new-game-project/CLAUDE.md`](new-game-project/CLAUDE.md) for full project
context — milestones, architecture, and the headless QA workflow.
